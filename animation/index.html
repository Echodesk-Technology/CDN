Customization

Animate Hover style

To change color of animate-hovered-style, add this to your css

.animate-hovered-style:hover {
color: yourColor !important;
}
.animate-hovered-style::after {
background-color: yourColor !important;
}

SlideInLeft Background

To change background color of animate-slideInLeftBg, add this to your css

.animate-slideInLeftBg {
background-color: background-image: linear-gradient( to left, transparent, transparent 50%, yourColor 50%, yourColor) !important;
}
